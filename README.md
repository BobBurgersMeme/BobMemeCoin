# BobMemeCoin

![BobMemeCoin logo](bobmeme.png)

BobMemeCoin is a meme coin and fan project inspired by the Bob's Burgers comic/cartoon universe.

The project is made for fun, community, and meme culture. It is not an official Bob's Burgers product and is not affiliated with the creators, owners, or rights holders of Bob's Burgers.
